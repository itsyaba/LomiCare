# Retreat Mode images

Drop real Kuriftu / resort / Ethiopian-nature photos here. The retreat page
(`/retreat`) renders each as a full-bleed background **over a warm gradient**,
so a missing file simply shows the gradient — nothing breaks before you add them.

Recommended: large landscape JPGs (~1600×2000 or wider), warm/natural tones.

| File              | Where it shows         | Suggested subject                                  |
| ----------------- | ---------------------- | -------------------------------------------------- |
| `hero.jpg`        | Top hero               | Wide resort / lake / sunrise establishing shot     |
| `morning.jpg`     | Moment 1 — Arrival     | First light, calm arrival, garden / lobby / water  |
| `activity.jpg`    | Moment 2 — Activity    | Spa, lake, walking path, gentle movement           |
| `nourishment.jpg` | Moment 3 — Nourishment | Ethiopian food, buna / coffee, a warm table        |
| `evening.jpg`     | Moment 4 — Evening     | Sunset, firelight, lanterns                        |
| `reflection.jpg`  | Moment 5 — Reflection  | Night sky, candle, quiet stillness                 |

`arrival.jpg` is reserved if you later want a photo behind the arrival form.
